# data-analysis-agent — 要件定義書

## 概要

CSV ファイルをアップロードし、ユーザーの分析リクエストに基づいて Python コードを生成・実行し、分析レポートを出力するエージェント。
Chapter 5 のコード生成→サンドボックス実行→レビューループパターンを採用する。

## 機能要件

### CSV アップロード
- ユーザーが CSV ファイルをアップロードする
- データの概要（カラム名、型、行数、基本統計量）を自動表示する

### 分析計画
- ユーザーの分析リクエストをサブタスクに分解する
- 分解結果をユーザーに表示する（Human Approval は省略可）

### コード生成・実行
- サブタスクごとに pandas / matplotlib のコードを LLM が生成する
- E2B サンドボックスで安全にコードを実行する
- 実行結果（stdout, グラフ画像, エラー）を取得する

### レビュー・リトライ
- 実行結果を LLM がレビューし、成功/失敗を判定する
- 失敗の場合はエラーを踏まえてコードを再生成する
- 最大 3 回リトライ

### レポート生成
- 全サブタスクの結果を統合して Markdown レポートを生成する
- グラフ画像はインラインで表示する

## 受け入れ条件

- [ ] CSV をアップロードして概要が表示される
- [ ] 分析リクエストからコードが生成・実行される
- [ ] E2B サンドボックスでコードが安全に実行される
- [ ] エラー時にリトライが発生する
- [ ] 分析結果がレポート形式で表示される
