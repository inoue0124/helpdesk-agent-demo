# user-inquiry-form — 要件定義書

## 概要

ユーザーがお問い合わせフォームから質問を送信し、回答を受け取る画面を構築する。

## 機能要件

### お問い合わせフォーム
- フォーム項目:
  - ユーザー名（任意、テキスト入力）
  - お問い合わせ内容（必須、テキストエリア）
- バリデーション: お問い合わせ内容は空欄不可
- 送信ボタンでフォームを送信する

### 受付完了表示
- 送信後に受付完了メッセージを表示する
- 受付番号（inquiry ID）を表示する
- 「担当者が確認後、回答いたします」のメッセージ

### 回答表示
- 回答が届いたら画面上に表示する
- ポーリング（5秒間隔）で回答の有無を確認する
- 回答表示後は「新しいお問い合わせ」ボタンでフォームに戻れる

## 非機能要件

- shadcn/ui + Tailwind CSS でスタイリングする
- モバイルフレンドリーなレスポンシブデザイン

## 受け入れ条件

- [ ] フォームから問い合わせを送信できる
- [ ] 空欄送信時にバリデーションエラーが表示される
- [ ] 送信後に受付完了メッセージと受付番号が表示される
- [ ] 回答が届いたら自動的に表示される
- [ ] 「新しいお問い合わせ」ボタンでフォームに戻れる
