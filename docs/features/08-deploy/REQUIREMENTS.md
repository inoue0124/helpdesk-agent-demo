# deploy — 要件定義書

## 概要

本番環境へのデプロイと公開準備を行う。

## 機能要件

### デプロイ
- Hetzner Cloud VPS にデプロイする
- Docker Compose (本番構成) でワンコマンド起動する
- Caddy で自動 HTTPS を設定する
- 初期データ投入を実行する

### ドキュメント・公開準備
- README.md にアーキテクチャ図、スクリーンショット、技術選定理由を記載する
- デモ動画 or GIF を作成する

## 非機能要件

| 項目 | 要件 |
|---|---|
| サーバー | Hetzner Cloud CAX21 (ARM / 4vCPU / 8GB / 80GB) |
| コスト | 月額約 7 EUR |
| HTTPS | Caddy の自動 Let's Encrypt |
| DNS | 独自ドメインを設定 |

## 受け入れ条件

- [ ] 本番環境で全サービスが正常に起動する
- [ ] HTTPS でアクセスできる
- [ ] ユーザー画面 → 問い合わせ → エージェント処理 → 回答の一連のフローが動作する
- [ ] README.md が整備されている
