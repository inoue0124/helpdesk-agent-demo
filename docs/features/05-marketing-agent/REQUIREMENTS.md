# marketing-agent — 要件定義書

## 概要

マーケティング課題に対して複数の専門エージェントが協調して施策を提案するエージェント。
Chapter 7 MACRS のマルチエージェント協調パターンを採用する。

## 機能要件

### 会話形式のインタラクション
- ユーザーがマーケティング課題を入力する
- エージェントが状況を深掘りする質問を行う
- ユーザーの回答に基づいて具体的な施策を提案する
- チャット形式で複数ターンのやり取りを行う

### マルチエージェント構成
- **Router Agent**: ユーザー入力を分類し、適切なサブエージェントに振り分ける
- **Question Agent**: 状況をヒアリングする深掘り質問を生成する
- **Recommendation Agent**: 具体的なマーケティング施策を提案する

### 会話管理
- 会話履歴を保持し、文脈を踏まえた応答を行う
- セッション単位で会話を管理する

## 受け入れ条件

- [ ] マーケティング課題を入力すると深掘り質問が返る
- [ ] 質問への回答後に具体的な施策提案が返る
- [ ] 会話の文脈が保持される
- [ ] Router が適切にエージェントを切り替える
